Simple Interest Calculator

This intuitive web application simplifies financial calculations by allowing users to quickly determine interest amounts on loans or investments. Built with HTML, CSS, and JavaScript, it features a clean interface where users input the principal amount, annual interest rate, and time period (in years/months). The system instantly computes simple interest using the formula I = PRT/100, displaying both the interest earned and total payable amount. Designed for financial literacy, it helps students understand basic finance concepts, assists small business owners in loan planning, and enables individuals to compare investment options. Additional features include a breakdown visualization chart, the ability to save calculations, and an option to toggle between yearly/monthly computations. The responsive design ensures accessibility across devices, making it equally useful for quick mobile calculations or detailed desktop analysis.

Real Time News App

This dynamic news aggregator platform curates and delivers live updates from multiple verified sources including major publications, blogs, and press releases. The React-based frontend with Node.js backend offers personalized news feeds based on user preferences and reading history. Key features include AI-powered categorization (politics, technology, sports etc.), customizable alerts for breaking news, and a bookmarking system for saving articles. The application implements sentiment analysis to highlight positive/negative tone in articles and offers a "fact check" badge for verified stories. Its purpose extends beyond information delivery to combating misinformation by cross-referencing sources and providing media bias indicators. The clean, ad-free interface prioritizes readability with adjustable text size and dark mode, serving as a comprehensive yet distraction-free news hub for professionals, researchers, and general readers.

Learning Management System

A comprehensive educational platform developed with MERN stack (MongoDB, Express, React, Node.js) that facilitates complete course management for institutions and corporate trainers. The system enables educators to upload multimedia content (videos, PDFs, quizzes), track student progress through analytics dashboards, and conduct live virtual classes with integrated Zoom API. Students benefit from interactive features like discussion forums, peer review systems, and AI-powered tutoring chatbots. The LMS supports SCORM-compliant content, gamification through badges/leaderboards, and accessibility features like screen reader compatibility. Its primary purpose is to bridge geographical educational gaps by providing a centralized, scalable solution for remote learning while reducing administrative overhead through automated grading, attendance tracking, and performance reporting tools for administrators.
